# 2주차 과제 — HTML 자기소개 페이지

## 📋 과제 내용

수업 중 AI 협업 미션(HTML 자기소개 페이지)에 대해 아래 내용을 완성해서 제출하세요.

- [ ] AI가 생성한 HTML 코드를 VSCode에 붙여넣기
- [ ] `<h1>` 태그 안의 이름을 본인 이름으로 수정
- [ ] `<p>` 태그에 나만의 소개글 2줄 이상 작성
- [ ] 수정된 파일을 GitHub에 commit & push 후 링크 제출

## ✅ 제출 방법

1. 이 Repo를 **Fork** 하세요
2. `submissions/본인이름/` 폴더를 만들고 `index.html` 파일 업로드
3. 완료 후 **Pull Request** 제출
4. PR 제목: `[제출] 홍길동 - 2주차 과제`
